# CLAUDE.md

> **Правила проекта находятся в [AGENTS.md](./AGENTS.md) — прочитай его перед любой задачей.**

`AGENTS.md` — **единственный источник истины** по стилю кода, архитектуре (DDD,
доменные алиасы, barrel-файлы), строгому TypeScript, реактивности и watcher-гигиене,
шаблонам Vue, именованию, стилям и обязательным проверкам перед коммитом
(`eslint . --fix` + `vue-tsc`). Его правила имеют приоритет над любыми
конфликтующими соглашениями.

Дополнительно:

- **Карта проекта и список фич** — в [PROJECT_MAP.md](./PROJECT_MAP.md).
- **Скиллы для ИИ-агентов** — в [.agents/skills/](./.agents/skills/); перед
  задачами по UI, архитектуре или новым инструментам их чтение обязательно
  (см. раздел «AI Agents Skills» в `AGENTS.md`).
- **Anti-Duplication Protocol** — перед созданием любого composable/утилиты
  сначала ищи готовое в `.nuxt/imports.d.ts` (см. «Smart Context Discovery» в
  `AGENTS.md`).
